# Кроссбраузерность

Основная часть сайтов должна поддерживаться в последних версиях таких браузеров, как:

## Десктоп

* Chrome
* Safari
* Firefox
* Edge
* Yandex Browser
* Opera

## Мобильные устройства и планшеты

* Chrome
* Safari

В определенных случаях данный список может меняться.

# Адаптивность

Для основной части сайтов используются два(три) базовых брейкпоинта:

* с 320(280) до 1024 (мобильные устройства и планшеты)
* с 1025 и выше (десктоп)

Остальные брейкпоинты добавляются по мере надобности, либо в зависимости от предоставленных макетов.

Сайт должен быть полностю адаптирован по ширине начиная от 320(280) и по высоте начиная от 550 пикселей.
